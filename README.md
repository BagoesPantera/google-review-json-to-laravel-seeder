# Google Review JSON to Laravel Seeder

This project converts Google review JSON (from APIvy) into a Laravel seeder PHP array.

**Purpose**: Receive review data from APIvy (or another Google-Places/Reviews JSON provider) and transform it into a PHP array suitable for use in a Laravel seeder.

## Prerequisites

- **Python**: 3.8+ recommended
- No external dependencies assumed; if a script requires packages, they will be listed in repository files.

## Repository Files (key items)

- `app.py` - main script (entry point)
- `app-import-from-json-file.py` - import JSON from a file and convert it
- `app-handle-null-stars.py` - optional helper to handle null/empty ratings
- `app-remove-under-4-star.py` - optional helper to filter out reviews under 4 stars
- `review.json` - example / input JSON (place your APIvy response here)
- `seeder_output.php` - example output generated by the scripts

## Quick Usage

1. Put the JSON response from APIvy into `review.json` (or save it as another filename).
2. Run the conversion script. Example commands:

`python app-import-from-json-file.py`

or, if `app.py` is the primary runner:

`python app.py`

3. The script will generate `seeder_output.php` (or print to stdout). That file contains a PHP array you can paste into a Laravel seeder.

**Example Input (simplified)**
```json
{
    "reviews": [
        {
            "title": "Tanah Merah Art Resort",
            "url": "google.com",
            "stars": 5,
            "name": "Rara",
            "reviewUrl": "google.com",
            "text": "Lorem"
        }
    ]
}
```

**Example Output (PHP seeder array)**
```php
<?php
    array (
        0 =>
        array (
            'id' => 1,
            'name' => 'Rara',
            'message' => 'Lorem',
            'rating' => 5,
            'review_url' => 'google.com',
            'created_at' => now(),
            'updated_at' => now(),
        ),
    )
```

**Notes**
- The scripts in this repository include helpers to filter or normalize reviews (for example, removing reviews below 4 stars or handling null ratings).
- If your JSON structure from APIvy differs, open the relevant script and adapt the parsing section accordingly.
